What are the Semantic tags?

Semantic HTML is the use of HTML markup to reinforce the semantics, or meaning, of the information in web pages and web applications rather than merely to define its presentation or look. Semantic HTML is processed by traditional web browsers as well as by many other user agents.

What are the differences between Flex and Grid?

Grid is made for two-dimensional layout while Flexbox is for one. This means Flexbox can work on either row or columns at a time, but Grids can work on both. Flexbox, gives you more flexibility while working on either element (row or column). HTML markup and CSS will be easy to manage in this type of scenario. GRID gives you more flexibility to move around the blocks irrespective of your HTML markup.

What are the differences between Bootstrap and Tailwind?

"Responsiveness" is the primary reason why developers consider Bootstrap over the competitors, whereas "Highly customizable" was stated as the key factor in picking Tailwind CSS.

What is CSS Box Model?

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, paddingg, and the actual content. The CSS box model refers to how HTML elements are modeled in browser engines and how the dimensions of those HTML elements are derived from CSS properties. It is a fundamental concept for the composition of HTML webpages.
